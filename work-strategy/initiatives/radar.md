# Radar

## What It Is
An internal analytics research and knowledge system that ingests data assets, dependencies, and investigation learnings to accelerate analyst workflows and reduce tribal knowledge.

## Primary Users
- Primary: data analysts
- Secondary: engineers, leaders

## Core Jobs
- Investigate data issues
- Research dashboards and dependencies
- Debug SQL/code
- Retrieve relevant data points
- Accelerate manual investigation workflows

## Strategic Value
- Converts individual investigations into reusable knowledge
- Reduces tribal knowledge
- Creates growing institutional memory via Git-based feedback loop

## Current State
- Internal alpha / exploration
- Installed on team machines and a few partner machines
- Parallel Git tenant migration underway

## Proven Value
- Audited 14,508 procs and generated 4,692 fixes in 4 hours
- Identified deep dependency chains for deprecated systems

## Current Risks
- InfoSec review not yet completed
- Git migration time crunch
- Reliability and hallucination risk
- Context window limitations

## Success (3–6 Months)
- Successful InfoSec review
- Stable post-migration setup
- Proven reliability metrics
- Expanded real-world use cases

## Explicit Non-Goals / Boundaries
- Not yet production-ready
- Not yet broadly rolled out
- Not yet fully governed or validated
